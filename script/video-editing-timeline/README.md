# Video Editing Timeline Demo

纯手写视频编辑时间轴原型 — 2 轨深灰色，白底简洁风。

## 功能

- 2 条深灰轨道，带左侧标签
- **100ms 刻度**：标尺最小刻度 0.1s，大字为主刻度
- **10ms 精度**：播放头位置精确到 0.01s，时间显示 `MM:SS.mmm`
- **拖拽无极移动**：鼠标按下拖拽播放头，松开即定位
- **播放 / 暂停**：按钮或 Space 切换，rAF 驱动实时走时，播放头自动跟随滚动
- 减速时 `← →` 10ms 步进，`Home / End` 跳首尾并暂停
- Ctrl+滚轮缩放（20～400 px/s）
- 吸附开关（Snap，100ms 对齐）
- `window.getCurrentTime()` 获取当前时间光标位置（秒）

## 运行

直接浏览器打开 `index.html`，零依赖。
